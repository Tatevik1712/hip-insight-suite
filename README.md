## О проекте

HipDx AI решает две задачи:

**Клиническую** — помогает врачу диагностировать дисплазию тазобедренного сустава (ДТС) у детей до 2 лет. Система автоматически находит анатомические точки на рентгеновском снимке, рассчитывает клинические параметры и классифицирует патологию.

**Образовательную** — позволяет студентам самостоятельно расставлять точки и сравнивать свой результат с предсказанием ИИ, понимая на основе чего система пришла к выводу.

## Возможности

### Режим врача
- Загрузка снимков: **PNG, JPG, DICOM**
- Автоматическое определение 6 анатомических точек нейросетью
- Бинарная классификация: **патология / норма** с уверенностью в %
- Расчёт клинических параметров:
  - Ацетабулярный угол α
  - Дистанции h и d по Хильгенрейнеру
  - Положение головки по Перкину
- Классификация степени дисплазии по Tönnis (I–III)
- Возрастные нормы 0–24 мес. с учётом пола
- Для DICOM — автокалибровка через `PixelSpacing`
- Сохранение данных пациента и результатов в PostgreSQL
- Галерея всех сохранённых снимков с фильтрацией

### Режим обучения
- Ручная расстановка 6 точек с пошаговой инструкцией
- Визуализация линий Хильгенрейнера, Перкина, касательных к крышке впадины
- Сравнение своих точек с предсказанием ИИ
- Пояснение каждого параметра и его клинического значения

## Быстрый старт

### Требования
- Node.js ≥ 18
- Python 3.11
- PostgreSQL 16

### 1. Фронтенд

```bash
npm install
npm run dev
# → http://localhost:8080
```

### 2. База данных

```bash
# Создать БД
createdb hipdx

# Применить схему
psql hipdx -f Back/schema.sql
```

### 3. ML-бэкенд

```bash
# Установка зависимостей Python
pip install flask flask-cors pydicom pillow numpy \
            tensorflow scikit-learn joblib requests psycopg2-binary

# Терминал 1 — нейросеть (порт 5000)
cd ML && python3 ml_server.py

# Терминал 2 — основной сервер (порт 5001)
cd Back && python3 dicom_to_png.py
```

> Без ML-бэкенда приложение работает в режиме ручной расстановки точек.

## Медицинская справка

Метод рентгенометрии по **Хильгенрейнеру** — стандарт диагностики ДТС у детей.

| Параметр | Норма | Патология |
|----------|-------|-----------|
| Ацетабулярный угол α | ≤ 28–30° (по возрасту) | > нормы → дисплазия |
| Дистанция h | 8–12 мм | < 8 мм → смещение вверх |
| Дистанция d | 10–15 мм | > 15 мм → латерализация |
| Линия Перкина | Медиально | Латерально → подвывих |

**Классификация по Tönnis:**

| Степень | Клиника |
|---------|---------|
| Норма | Все параметры в норме |
| I (предвывих) | Увеличен α, центрирование сохранено |
| II (подвывих) | Увеличены α и d, h в норме |
| III (вывих) | Увеличены α и d, уменьшена h |

> ⚠️ Система является вспомогательным инструментом. Окончательный диагноз ставит врач.
